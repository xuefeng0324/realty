# v0.45.0 · 社区 总价 × 单价 双轴散点 (trend-25)

**主题**：每社区中位单价 × 中位总价散点图 + 4 象限划分,让用户一眼看出板块价值定位。

## 数据源
- `static/seed/listings.csv` (per community 派生 median)
- `static/seed/communities.csv` (社区名/区名)

## 注意
- `total_price_10w` 字段 100% 空, 改为计算: `total = unit_price × area_sqm / 10000`

## 公式
```
per (city, community):
  count = listings 命中数 (≥3)
  median_unit_price = median(unit_price)
  median_area = median(area_sqm)
  median_total_price_10w = median(unit_price × area / 10000)
  area_cohort = if a < 60: 小户型 else if a < 110: 改善 else 大户型
  
  city_median_unit = median of all city points
  city_median_total = median of all city points
  
  quadrant = (high_up, high_tp) ? 豪宅板块
           : (high_up, !high_tp) ? 学区刚需
           : (!high_up, high_tp) ? 改善低密
           : value: 价值洼地
```

min_count=3 过滤 → 24 社区散点 (3 城 × 8/11/5)

## 业务价值
- 客户选区域: 直接看散点知道自己定位
- 投资决策: 学区刚需区(小户型高单价) 是流动性最好的资产
- 跨城对比: 同样的"豪宅板块", 深圳 800-1100 万 vs 珠海 300-400 万 (3 倍差距)
- 视觉震撼: SVG 散点图 是 dashboard 唯一 "高密度信息图", 与之前的卡片型 UI 形成差异化
