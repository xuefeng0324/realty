# v0.49.0 topnav-1 周期 sticky 切换

**release date:** 2026-07-14

**summary:** dashboard 顶部新增 sticky 周期切换条, 用户不用滚到顶部 picker 也能跳周次了。

## 设计

```
┌─────────────────────────────────────────────┐
│  📅 第 26 / 26 周 · 2026-06-28              │  ← 黑底蓝灰渐变
│  ┌────────────────┬────────────────┐         │
│  │  ‹ 上一周      │  下一周 ›       │  ← 半透明白按钮
│  └────────────────┴────────────────┘         │
└─────────────────────────────────────────────┘
```

- **深色 sticky 顶栏** (`#1e293b → #334155` 渐变, `border-radius: 16rpx`)
- **大号黄色周次编号** (`#fbbf24`, 等宽字体): "26" 突出
- **当前周期 + 总周数** + ISO 日期
- **左右切换按钮**: 禁用时 0.35 透明度, `pointer-events: none`
- **`position: sticky; top: 0; z-index: 50`**: 滚动到底部仍可见

## 实现

```ts
const currentPeriodIdx = computed(() =>
  periods.value.indexOf(app.weekEnd)
);

function stepPeriod(delta: number) {
  const list = periods.value;
  const cur = currentPeriodIdx.value;
  const next = cur + delta;
  if (next < 0 || next >= list.length) return;
  app.setWeekEnd(list[next]);
  loadRankingAndDistrict();
  showToast(`已切到 ${list[next]}`);
}
```

触发 `app.setWeekEnd()` 后, `watch(() => [app.cityId, app.weekEnd])`
自动重新 `loadAll()`, 全部 26 张卡刷新。
