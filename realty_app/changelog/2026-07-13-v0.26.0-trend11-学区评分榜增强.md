# v0.26.0 trend-11 学区评分小区榜增强

## 新功能

dashboard 「学区评分 Top 小区」卡新增 3 组交互式控件：

1. **区过滤 (chips)**: 该城市所有出现过的区名，多选 (再次点击取消)
2. **最低评分 chips**: 不限 / 70+ / 75+ / 80+ / 85+
3. **排序 chips**:
   - **评分** (avg_school_score 降序，默认)
   - **均价** (median_unit_price 升序 — 最便宜)
   - **挂牌** (listing_count 降序 — 货量最大)
   - **校数** (school_count 降序)

数据来源不变：`school_premium_community.csv` (52 行：深 39 / 广 8 / 珠 5)。

## 数据流

1. **store.ts**: `SchoolPremiumCommunitySort` 类型 + `getSchoolPremiumCommunityRank()` 增加 `minScore / districtFilter / sort` 参数
2. **queries.ts**: `getSchoolPremiumCommunityRank()` 透传新参数
3. **dashboard.vue**:
   - `spDistrictFilter / spMinScore / spSort` 三个 ref
   - `spDistrictOptions / spMinScoreOptions / spSortOptions` 控件选项
   - `toggleSpDistrict(d)` 切换区多选
   - `spSortLabel` computed 当前排序文案
   - 切城市时重置过滤 (避免区跨城污染)
   - 控件变化触发 `watch` 重新加载该卡 (无需重新 loadAll)

## UI

3 行 chip 控件：
```
[区]  南山区  福田区  罗湖区  ...
[最低评分]  不限  70+  75+  80+  85+
[排序]  评分  均价  挂牌  校数
```

选中状态：青色背景 + 白字；未选：深灰底 + 灰字。
