# v0.15.0 - 地铁规划 overlay
> 2026-07-12

## 一句话
map-view 新增「地铁规划」模式：把 metro_planning.csv 21 条规划/在建地铁线画到地图（绿=即将开通 / 橙=在建 / 灰=规划）。点 marker 显示线路详情（站点数 / 长度 / 预计开通年）。

## 数据基础
`static/seed/metro_planning_geo.csv` (21 行)：
- start_lat / start_lng / end_lat / end_lng：高德 POI text search 拿坐标
- start_confidence / end_confidence：high (匹配名称含查询关键词 / 站名) / medium / manual (手填) / missing
- 21/21 行可画 polyline (drawable lines = 21)

```
深圳 15 行 (深圳五期/四期)
广州 4 行 (广州三期调整/四期)
珠海 2 行 (珠肇高铁 / 南珠城际)
```

## 改动
### 数据脚本
- `scripts/crawl_amap_metro.py`：调高德 `/v3/place/text`，给每条地铁线的 start_station / end_station 拿 lat/lng + formatted_address + confidence
- `scripts/enrich_metro_geo_manual.py`：补 missing 站点的近似坐标（基于公开地理信息），21/21 全可画

### 数据文件
- `static/seed/metro_planning_geo.csv`（新增，21 行 × 14 列）

### 数据层
- `src/local/types.ts`：`LocalMetroLineGeo` 接口；DataSnapshot 加 `metroLineGeos: LocalMetroLineGeo[]`
- `src/local/importer.ts`：SnapshotInputs 加 `metroPlanningGeoCSV`，importSnapshot 解析 CSV
- `src/local/store.ts`：`getMetroLineGeos()` / `getMetroLineGeosByCity(cityId)`
- `src/local/queries.ts`：`MetroLineGeoItem` / `MetroLinesGeoResponse` + `getMetroLineGeos({cityId})`
- `src/local/seedSnapshot.ts`：默认加载 metro_planning_geo.csv
- `src/local/dataRefresher.ts`：merge 时保留 `existing?.metroLineGeos`
- `src/pages/settings/settings.vue`：csv-url 模式拉取 metro_planning_geo.csv

### UI (`src/pages/map-view/map-view.vue`)
- `MapMode` 加 `metro` 模式
- 5 模式轮换：count → price → listings → poi → metro → count
- metro 模式：用 uni-app `<map>` 的 `polyline` 属性画起点→终点直线（高德 H5 用 canvas 渲染）
- 颜色按 status：`#22c55e`(即将开通) / `#f59e0b`(在建) / `#94a3b8`(规划)
- 每条线 2 个 marker（start/end），点击显示 line info-card
- info-card 显示：线路名 / status / 预计开通年 / 起点↔终点 / 站点数 / 长度

## 测试
