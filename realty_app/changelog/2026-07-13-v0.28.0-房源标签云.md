# v0.28.0 new-6 房源 tags 标签云

## 新功能

dashboard 新增「🏷️ 房源标签云 · {城市}」卡片，展示当前城市最热门的房源 tags。

### 派生规则 (基于 listings.csv + 周边数据)

| 维度 | 标签 |
|---|---|
| 户型 | 一房 / 两房 / 三房 / 四房 / 大户型 (≥150㎡) |
| 价格 | 笋盘 (低于 city 中位 70%) / 高价 (高于 1.5×) |
| 朝向 | 朝南 / 南北通透 |
| 装修 | 豪装 / 精装 / 简装 / 毛坯 |
| 学区 | 名校区 (学区评分 ≥ 80) |
| 地铁 | 近地铁 (≤500m) / 地铁可达 (≤1500m) |
| 楼龄 | 楼龄新 (≥2015) / 老破小 (≤2000 且 <70㎡) |
| 楼层 | 高楼层 (≥20 层) |
| 电梯 | 带电梯 |
| 平台 | VR房源 / 随时看房 / ... (来自 listings.tags_json) |

字号按 count 分 5 档（5=最热，1=最冷），颜色由深灰渐变到青蓝。

## 数据流

1. **compute_listing_tags.py** 派生 tags + 输出：
   - `listing_tags.csv` (7517 行：listing_id × tag 一行一对)
   - `listing_tags_summary.csv` (52 行：city × tag × count × share)
2. **importer.ts** `parseListingTags()` → `LocalListingTag[]`
3. **store.ts** `getListingTagsByCity(cityId)`
4. **queries.ts** `getListingTagCloud({ cityId, limit })` → `TagCloudResponse`
5. **dashboard.vue** 「🏷️ 房源标签云」卡，5 档字号 + 点击 tag 显示提示
