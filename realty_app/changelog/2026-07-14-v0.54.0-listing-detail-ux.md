# v0.54.0 — detail-1 listing 详情页 UX 优化

## 变更

### 1. `src/pages/listing-detail/listing-detail.vue` — 顶部快捷导航
新增 `.quicknav` 4 按钮条:

| 按钮 | 行为 |
|---|---|
| ← 返回 | `uni.navigateBack` 或 fallback `switchTab` → dashboard |
| 📊 仪表盘 | `uni.switchTab` → `/pages/dashboard/dashboard` |
| 🏘️ 小区详情 | 跳到 `community/community?id={community_id}` |
| 🔁 同小区其他 (N) | 主按钮 (紫渐变), 显示同小区其他 listings 数 |

### 2. 同小区其他在售卡片
新增「🔁 同小区其他在售 · {communityName}」卡片:
- 数据源: `store.getListingsByCommunity(communityId)` 复用 listings snapshot
- 过滤掉当前 listing 自身 + 无单价的
- 按 `unitPrice` 降序排序, 取 Top 10
- 横向对比显示: 户型/面积/朝向/装修 + 总价 + 单价
- 每行 `tap-row` hover + `@click="goListing(id)"` → `uni.redirectTo` 到新 listing 详情
- 底部提示语: 同小区同户型差价 ±5-10% 属正常议价空间

### 3. 新增导航函数
```ts
function goBack() {
  // 智能回退: navigateBack 优先, fallback switchTab dashboard
}
function goDashboard() {
  uni.switchTab({ url: "/pages/dashboard/dashboard" });
}
function goListing(id: number) {
  uni.redirectTo({ url: `/pages/listing-detail/listing-detail?id=${id}` });
}
```
