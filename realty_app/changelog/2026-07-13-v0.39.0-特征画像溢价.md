# v0.39.0 · 特征画像溢价 (trend-19)

**主题**：基于 listings.csv，按"城市 × 维度 × 桶"算出 **premium%** —— 哪些特征让房子更贵/更便宜。

## 数据源
- `static/seed/listings.csv` (1286 行)
- `static/seed/cities.csv` (city_name lookup)

## 公式
```
premium% = (bucket_median_unit_price ÷ city_median_unit_price - 1) × 100
```

## 4 维度 / 桶
- **bedrooms** (户型): 1室/2室/3室/4室/5室
- **area_sqm** (面积): <50 / 50-80 / 80-110 / 110-150 / 150-200 / 200+
- **orientation** (朝向): 南 / 东南 / 南北通透 / 西 / etc.
- **decorate** (装修): 毛坯 / 普装 / 简装 / 精装 / 豪装

## dashboard 新卡
**「💎 特征画像溢价 · {city}」**:
- 4 个 dim block (户型/面积/装修/朝向)，每 block 显示 top 3 桶
- 每行：bucket 名称 + 横向条形图 (宽 = |premium|/30) + 颜色 (红↑/蓝↓/灰平) + 百分比
- 阈值：±1% 区分 (广州 ±5% 内波动，阈值 1% 适合)
- `minCount=5` 过滤小样本桶
