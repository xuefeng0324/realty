# v0.14.0 - 学区评分 Top 小区

> 2026-07-12

## 一句话

dashboard 新增「学区评分 Top 小区」卡：按 (city_id, avg_school_score) 降序展示该城市里沾名校光最多的小区（含区名 + 评分 + 学校数 + 中位单价）。和 v0.11.0「学区溢价榜」互补：前者是区维度，后者是小区维度。

## 改动

### 数据层

- `src/local/store.ts`: `getSchoolPremiumCommunityRank({cityId, minListings, limit})`
  - 过滤 `school_count >= 1, avg_school_score > 0, listing_count >= minListings`
  - 排序：先按 score 降序，并列按 median_unit_price 降序
- `src/local/queries.ts`: `getSchoolPremiumCommunityRank` 返回 `SchoolPremiumCommunityOverview`
  - 接口 `SchoolPremiumCommunityItem { communityId, districtName, communityName, schoolCount, avgSchoolScore, listingCount, medianUnitPrice, rank }`

### UI

- `src/pages/dashboard/dashboard.vue`: 新增「学区评分 Top N 小区」卡
  - 金/银/铜牌 + 学校评分 + 学校数 + 中位单价
  - 点击小区行 → 跳转 community 详情
  - 与「小区排行」(综合评分) 互补

## 测试

- `tests/buildIntegrity.test.ts` 新增 5 个测试（queries/store 函数 + csv 完整性 + 评分范围）
- `tests/e2e/smoke_school_community.mjs`（新增）：广州/深圳切换 + 截图
- 全量：198/198 单测通过（+5），13/13 smoke 全绿
