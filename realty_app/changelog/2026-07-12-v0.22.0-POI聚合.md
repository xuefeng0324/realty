# v0.22.0 — POI marker 聚合 (map-3)

## 背景
原 map-view POI 模式每类限 25 个 = 总共最多 125 个 marker 显示，丢了 553 个。
现 poi_seed.csv 有 678 个 POI (subway 107 / school 142 / hospital 143 / mall 144 / park 142)。

## 改动
- 复用 `cluster.ts` (v0.18.0) 算法 → POI marker 网格聚合
- **每类单独 cluster**（避免不同类 POI 混合成一颗大球）
- 单 POI → 圆形彩色图标 (emoji + 类别色背景) + click 弹出 info-card
- 聚合 POI → 大号彩色气泡 + 数字 (1/3/5+) + click 放大到下一 zoom 让 cluster 拆分
- 自适应 zoom：zoom 11 (城市级) 聚合多，zoom 16+ 几乎不聚合 (cell ≈ 250m)
- SVG data URI inline 图标（无外链依赖，uni-app H5 兼容）
- onMarkerTap 加 POI cluster 路径 (`markerId <= -1000000` → zoom+1)
- POI 模式 legend 文案更新
