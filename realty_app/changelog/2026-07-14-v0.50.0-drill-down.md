# v0.50.0 drill-1 小区 drill-down

**release date:** 2026-07-14

**summary:** dashboard 上的小区散点 + 行政区图 marker 之前只能看不能点;
现在点这些元素直接跳到 `/pages/community/community?id={communityId}`,
显示该小区详情 (户型 / 挂牌 / 学校 / 地铁规划 / 5 维评分)。

## 3 个可点击入口

### 1. 散点卡 quadrant row (4 象限 × Top 3 = 12 行)
- 整个 `<view class="scatter-row tap-row" hover-class="tap-row--active">` 点击
- 末尾加 `›` 视觉提示
- hover 浅色背景高亮

### 2. 散点 SVG 圆点 (8+ 个)
- `<circle class="scatter-pt" @click="goCommunity(p.communityId)" />`
- `<title>` 提供原生气泡提示 (社区名 · quadrant)
- hover: 透明度从 0.55 → 0.95, 黄色描边加粗

### 3. 行政区图 marker (≤30 时命名 g / >30 时 bare circle)
- 命名 g: `<g class="map-marker-g tap-row" @click>` 整组点击 (文字+圆点)
- bare: `<circle class="map-marker-bare tap-row" @click>` 圆点本身
- hover: 红色填充 (#ef4444)

## 复用现有函数

`goCommunity(id)` 早就存在 (v0.11.0 通勤榜时实现):
```ts
function goCommunity(id: number) {
  uni.navigateTo({ url: `/pages/community/community?id=${id}` });
}
```
