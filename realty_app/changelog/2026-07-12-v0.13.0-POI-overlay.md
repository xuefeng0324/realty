# v0.13.0 - POI Overlay (地图周边配套图标)

> 2026-07-12

## 一句话

map-view 新增第四种模式「POI overlay」：把 poi_seed.csv 的 5 类 POI (🚇地铁 / 🏫学校 / 🏥医院 / 🛍商场 / 🌳公园) 画到地图上，5 类 toggle 自由开关，点击 POI marker 显示距离 + 类型 + 所属小区。

## 数据基础

`static/seed/poi_seed.csv` 现有 678 行 POI 数据：
- subway: 107
- school: 142
- hospital: 143
- mall: 144
- park: 142

## 改动

### 数据层

- `src/local/store.ts`: `getPoisByCity(cityId)` — 返回该城市所有 POI，按 communityId 关联 communities.csv 的 cityId
- `src/local/queries.ts`: `getCityPois({cityId, category?})` — 返回 `CityPoisResponse {cityId, total, counts, items}`

### UI

- `src/pages/map-view/map-view.vue`:
  - `MapMode = "count" | "price" | "listings" | "poi"`
  - 4 模式轮换：count → price → listings → poi
  - POI 模式下显示 5 类 toggle 按钮（带类别计数）
  - `poiMarkers` computed：每类取最近 25 个 POI（防 marker 过载），按 category 着色（地铁蓝/学校绿/医院红/商场橙/公园深绿）
  - POI info-card：选中 marker 后显示 POI 名 + 类型 + 距离 + 所属小区 ID + 地址

## 测试
