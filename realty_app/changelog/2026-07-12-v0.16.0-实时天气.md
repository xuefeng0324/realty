# v0.16.0 - 实时天气 + 4 天预报
> 2026-07-12

## 一句话
dashboard 新增「实时天气」卡：用高德 `/v3/weather/weatherInfo` 拿 3 座城市的实况天气 + 未来 4 天预报；含天气 emoji / 湿度 / 风力 / 粗略 AQI 估算。

## 数据源
- 高德 `/v3/weather/weatherInfo`
  - `extensions=base` → 实况: weather / temperature / winddirection / windpower / humidity / reporttime
  - `extensions=all` → 预报: 4 天 casts (dayweather / nightweather / daytemp / nighttemp / daywind / nightpower 等)

```
深圳 440300 / 广州 440100 / 珠海 440400
```

## 数据
`static/seed/weather.csv` (6 行 = 3 城 × 2 类型):
- live 行: weather / temperature / winddirection / windpower / humidity
- forecast 行: forecast_json 字段含 4 个 cast (JSON 字符串)
- 注: 高德 API 不含 AQI, 用 `湿度 + 风力 + 温度` 粗略估算

## AQI 估算规则 (粗略, 仅演示)
- 风力 >= 5 级 → 优
- 湿度 >= 85% 且风力 <= 2 → 轻度污染 (闷热)
- 温度 >= 35°C 且湿度 >= 60% → 轻度污染 (高温闷热)
- 湿度 >= 80% → 良
- 其它 → 优
- 生产环境请接 AQICN 或国控站 API 拿真实 AQI
