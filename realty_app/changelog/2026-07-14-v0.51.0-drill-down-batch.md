# v0.51.0 — drill-2 批量 ranking 行 drill-down

延续 v0.50.0 的 drill-down 思路: 把 dashboard 上 **未绑定跳转** 的 5 张 ranking 行卡片批量加上 `goCommunity` 点击跳转, 配合 `tap-row` hover 反馈, 进一步打通 Dashboard → 小区详情 的路径.

## 变更

### 1. `src/pages/dashboard/dashboard.vue` — 5 张卡片批量加 `@click`
所有 ranking 行的 `<view>` 增加 `class="*row tap-row"` + `hover-class="tap-row--active"` + `@click="goCommunity(it.communityId)"`:

| 卡片 | 行 class | 行数 (广州默认) | 跳转目标 |
|---|---|---|---|
| 🚇 地铁通勤榜 (`metroWalk`) | `.mw-row.tap-row` | 5 | `community/community?id=…` |
| 🚇 地铁规划受益 (`metroBenefit`) | `.mb-row.tap-row` | 8 | `community/community?id=…` |
| 🏪 生活配套评分 (`lifeConvenience`) | `.lc-row.tap-row` | 8 | `community/community?id=…` |
| 🏅 小区综合评分 (`communityScore`) | `.cs-row.tap-row` | 8 | `community/community?id=…` |
| 📅 房源新鲜度 (`listingFreshness` 新挂牌 + 滞销) | `.lf-row.tap-row` | 16 | `community/community?id=…` |

复用已有的 `goCommunity(id)` (uni.navigateTo 至 `/pages/community/community?id=…`).

### 2. `tap-row` hover 样式 (v0.50.0 已存在, 复用)
```scss
.tap-row {
  cursor: pointer;
  transition: background 0.15s;
}
.tap-row--active {
  background: rgba(99, 102, 241, 0.08);
}
```
鼠标 hover 时背景变浅紫, 与原 `tap-target`/`row-active` 区分 (旧的是 opacity, 新的是 background tint).
