# v0.53.0 — macro-1 LPR + 房贷利率

新增 dashboard 「💰 LPR + 房贷利率」卡片: 央行 PBOC 公开公告 LPR 1Y/5Y 历史 + 一线城市首套/二套房贷利率. 83 月数据 (2019-08 → 2026-06), 展示当前利率、累计下调、近 12 月走势 sparkline.

## 数据源

- **央行 PBOC 公开公告** (`http://www.pboc.gov.cn/`): LPR 改革 (2019-08-20) 以来每月 20 日 (遇节假日顺延) 公布的 1Y / 5Y+ LPR.
- 数据手工录入 (官方公告 + 主流财经媒体复核), 通过 `scripts/compute_lpr_history.py` 派生.
- 一线城市首套/二套加点规则:
  - **首套** = 5Y LPR - 30bp (一线城市普遍下调, 深/穗 2024 后普遍执行)
  - **二套** = 5Y LPR + 35bp (限购城市加严)

## 变更

### 1. `scripts/compute_lpr_history.py` (新)
- 录入 2019-08 → 2026-06 共 83 月 LPR 数据.
- 输出 `static/seed/lpr_history.csv`:
  - `month` (YYYY-MM)
  - `lpr_1y`, `lpr_5y` (%)
  - `mortgage_first`, `mortgage_second` (%)
  - `source` (PBOC 公开公告)
- 控制台输出累计下调: **5Y -1.35 pp** (4.85→3.50), **1Y -1.25 pp** (4.25→3.00).

### 2. `src/local/types.ts`
- 新增 `LocalLprRow` interface (month, lpr1y, lpr5y, mortgageFirst, mortgageSecond, source).
- `DataSnapshot` 新增 `lprHistory: LocalLprRow[]` 字段.

### 3. `src/local/importer.ts` / `store.ts` / `queries.ts`
- `parseLprHistory()` 解析 CSV → `LocalLprRow[]`.
- `store.getLprHistory()` 取全部 LPR.
- `queries.getLprOverview()` 返回 `LprResponse`:
  - `series` 全部月度升序
  - `latest` 最近一月
  - `cumDrop1y` / `cumDrop5y` 自 2019-08 累计下调 (pp)
  - `yoyDrop1y` / `yoyDrop5y` 近 12 月变化 (pp)

### 4. `src/pages/dashboard/dashboard.vue` — 新卡片
**卡片内容** (`data-tab="all,price"`):
- **4 个 KPI**: 1Y LPR / 5Y LPR (高亮) / 首套房贷 (绿) / 二套房贷.
- **累计下调**: 自 2019-08 + 近 12 月 双视角 (绿=下行, 红=上行).
- **5Y LPR sparkline**: 近 36 月 SVG 折线 + 末尾红点 (含 hover tooltip 显示月份+值).
- **底部说明**: 100w 贷 30 年每 25bp 约影响月供 ¥150.

### 5. 联动集成
- `settings.vue` 增加 `lprHistoryCSV` 远程拉取.
- `seedSnapshot.ts` import `lpr_history.csv?raw` 内嵌到默认快照.
- `dataRefresher.ts` 在重刷时保留旧 LPR 数据.
