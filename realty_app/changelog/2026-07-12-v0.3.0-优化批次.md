# v0.3.0 优化批次 (2026-07-12)

**类型收敛 + 暗色主题一致性 + a11y 改进**

本批次为 v0.3.0 主提交之后的优化迭代，未改版本号，所有改动向后兼容。

## 变更摘要

### 1. 类型安全：收紧 `any` 入参

- `src/api/listings.ts`：`filterListings(body: any)` → `filterListings(body: ListingFilterRequest)`，`apiPost<...>` 也加显式泛型
- `src/local/queries.ts`：`filterListings(req: any)` → `filterListings(req: ListingFilterRequest)`（接口在 `contracts.ts` 已存在，未重复定义）
- `src/pages/community/community.vue`：调用点去除误传的 `weekEnd` 字段（旧 `ListingFilterRequest` 不包含此参数，是 `getListingDetail` 用的）；`cityId` 由 `app.cityId || undefined` 收紧为显式 `null` 早返回，避免把 `undefined` 赋给必填 `number` 字段触发 TS2322
- 现有 `ListingFilterRequest` 接口已包含 `filters?: ListingFilterFilters` 与 `sort?: { field?, direction? }` 完整字段，无需扩展

### 2. 暗色主题一致性

所有页面已统一引用 `App.vue` 全局主题色 palette（`#0b1020` page / `#111827` card / `#1f2937` border / `#1e293b` panel），但仍有散落硬编码：

- `pages/community/community.vue`
  - `.chart-track` `background: #1f2937` → `#1e293b`（该元素是进度条底色，应为 panel 色）
  - `.bin-track, .dim-track` 同上
  - `.listing-row` border `1rpx solid #1f2937` 保持不变（palette 已是该值）
- `pages/wangqian/wangqian.vue`
  - `.metric-cell` `background: #0b1220` → `#111827`
  - `.wq-table-row` `border-bottom: 1rpx solid #0f172a` → `#1f2937`
  - `.sheet` `background: #0f172a` → `#111827`
- `pages/listing-filter/listing-filter.vue`
  - `.sheet` `background: #0f172a` → `#111827`
  - `.thumb-bubble`（tooltip）保持 `#0f172a`：tooltip 设计上需比 card 更暗以拉开层次
- `pages/listing-detail/listing-detail.vue`
  - `.explain-box` `background: #0f172a` → `#111827`
- `pages/dashboard/dashboard.vue`
  - `.sheet` `background: #0f172a` → `#111827`
  - `.stats70-cell` `background: #0b1220` → `#111827`

收尾后 grep 残留：仅 `.thumb-bubble` 一处保留（设计需要更深），其余统一。

### 3. 可访问性 (a11y) 改进

- `src/App.vue` 全局新增工具类
  - `.tap-target`：`min-height: 88rpx`（750rpx 设计稿下 44pt，符合 Apple HIG / WCAG 2.5.5）
  - `.focusable:focus, button:focus, .btn:focus`：键盘焦点环 `2rpx solid #4ade80`
  - `.sr-only`：屏幕阅读器专用隐藏文本（保留视觉空白）
  - `.card-active, .row-active`：按压反馈 `opacity: 0.75; background: #1e293b`
- `pages/listing-filter/listing-filter.vue` `.listing-row`：加 `min-height: 88rpx`
- `pages/community/community.vue` `.listing-row`：加 `min-height: 88rpx`
- `pages/dashboard/dashboard.vue` 给主要可点击 card/行加 `role="button"`、`tabindex="0"`、`hover-class`：
  - `.stats70-card`（全国 70 城卡）
  - `.wangqian-card`（政府网签卡）
  - `.community-row`（小区排行行）
  - `.sheet-item`（选择弹层项）

未做但保留作为后续：

- 4 个 `<image>` 总数 = 0（项目内无图片），无 alt 缺失问题
- tabBar 仅文字缺图标：项目原本设计如此，本次不动
