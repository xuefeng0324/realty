# v0.47.0 school-4 学区指标加权细分

**release date:** 2026-07-13

**summary:** 把每所学校的 5 个原始指标 (评级分 / 集团实力 / 区域均衡 / 趋势) 拆开看,
而不是混在一个综合分里。在 dashboard 上新增「学区 5 维评分」卡片。

## 数据源

`static/seed/school_indicators.csv` 已有 5 列原始指标 (不动):

| 字段 | 含义 |
|------|------|
| `latest_level_score_raw` | 当前评级分 0-100 |
| `group_school_flag_raw` | 是否集团校 (True/False) |
| `group_school_strength_raw` | 集团校实力 0-100 |
| `district_balance_level_raw` | 区域教育均衡 0-100 (本区 vs 全市均值) |
| `trend_delta_raw` | 评级变化趋势 (正=上升, 负=下降) |

**新文件**: `static/seed/school_dimensions.csv` — 城市内百分位排名 + 综合分

## 新脚本

`scripts/compute_school_dimensions.py`

- 对每个城市内的所有学校, 在 5 个维度上各算 rank (1 = 最好), 转成 0-100 百分位
- `composite_score = 40% 评级 + 20% 集团实力 + 15% 区域均衡 + 10% 趋势 + 5% 是否集团`
- 输出 58 行 (3 城市合计)

## 新类型 / parser / store / queries

- `LocalSchoolDimension` (types.ts)
- `parseSchoolDimensions` (importer.ts)
- `getSchoolDimensions` / `getSchoolDimensionsByCity` (store.ts)
- `SchoolDimResponse` + `getSchoolDimensions` (queries.ts): 5 个维度的 Top 10

## 新 dashboard 卡片

「🏫 学区 5 维评分 · {city}」

包含 2 个区块:

1. **🏆 综合 Top 5** — 横向 grid + 彩色进度条 (绿/蓝/黄/红按分数)
2. **📊 各维度最强 Top 3** — 2x2 grid:
   - 📈 评级 levelScore
   - 🏢 集团校实力 groupStrength
   - ⚖️ 区域均衡 districtBalance
   - 🚀 涨幅 trendDelta (绿/红色文字)
