# v0.29.0 trend-13 区房价指数

## 新功能

dashboard 新增「📈 区房价指数 · {城市}」卡片：

- 每个区显示：区名、最新周指数、WoW (周环比)、YoY (同比)
- 右侧 sparkline 柱状图：12-30 周指数走势
- 指数排序：按最新周 index_value 降序
- 颜色编码：≥110 红 / 100-110 灰 / 90-100 灰 / <90 绿
- 显示前 6 区 (深圳 6, 广州 4, 珠海 2)

## 数据流

1. **compute_district_index.py** 基于 `district_trend.csv` 计算指数：
   - `baseline = 各区最早 4+ listings 的周中位价`
   - `index_value = current_median / baseline * 100`
   - `mom_change = (current - prev_week) / prev_week × 100%`
   - `yoy_change = (current - 364 天前) / yoy × 100%`
2. 输出 `district_index.csv` (266 行 / 12 个区)
3. `LocalDistrictIndex` 类型 + `getDistrictIndex()` query
4. `districtIndex` ref + sparkline 柱状图 UI
