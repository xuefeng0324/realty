# v0.33.0 — trend-15 小区综合评分 (生活+学区+通勤 加权)

**日期**: 2026-07-13
**关键词**: 综合评分 / 多维加权 / 金银铜牌 / 一图看懂"最适合自住"

---

## 1. 背景

v0.31.0 起已有生活便利度卡 (6 维) 和学区评分卡 (单分)，但**没有一个"综合"分数**告诉用户"哪个小区最适合自住"。本版本把 3 类分数合成 1 个 0-100 的综合分 (生活50% + 学区30% + 通勤20%)，dashboard 新增「🏅 小区综合评分 Top 小区」卡片。

## 2. 加权公式

```
total_score = life_score × 0.5
            + school_score × 0.3
            + commute_score × 0.2

commute_score = {
  minutes ≤ 30:   100
  minutes = 60:   50
  minutes ≥ 120:  0
  // 30-60 线性 100→50; 60-120 线性 50→0
}
```

## 3. 数据源

复用现有 3 个 CSV，无需新抓数据：
- `life_convenience.csv` (score100, 0-100)
- `school_premium_community.csv` (avg_school_score, 0-100)
- `commute.csv` (transit_minutes, 0-120)

## 4. 输出

`community_score.csv` (52 行 / 3 城全覆盖)

| 字段 | 说明 |
|------|------|
| community_id, city_id, district_name, community_name | 标识 |
| life_score (0-100) | 生活便利度 |
| school_score (0-100) | 学区评分 |
| commute_minutes | 通勤分钟 (无则 null) |
| commute_score (0-100) | 通勤分 |
| **total_score (0-100)** | **综合分** |
| **rank_city** | **该城内排名 (1-based)** |
