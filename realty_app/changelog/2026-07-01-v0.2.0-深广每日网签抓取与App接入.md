# v0.2.0 (2026-07-01)

**新增功能 — 深广每日网签抓取与 App 接入**

### 变更摘要

- 新增 `scripts/crawl_daily_wangqian.py`：抓取深圳住建局 fdc 子站、广州住建商品房签约日更 API
- 输出 `static/daily_wangqian.csv`（城市/行政区 × 新房/二手 × 套数/面积）
- 新增 `src/local/dailyWangqian.ts`：App 端解析与查询，启动时 `?raw` 加载
- Dashboard 与 stats70 页展示「政府每日网签」卡片
- 新增 GitHub Actions `crawl-daily-wangqian.yml`：工作日定时 `fetch --merge` 并提交 CSV
- 新增单元测试 `tests/dailyWangqian.test.ts`

### 数据源

| 城市 | 接口 | 类别 |
|------|------|------|
| 深圳 | `fdc.zjj.sz.gov.cn` `getYsfCjxxGsDataNew` / `getEsfCjxxGsDataNew` | 新房 + 二手 |
| 广州 | `zfcj.gz.gov.cn` `mrxjspfqyxx.ashx` | 新房住宅签约 |

### 已知局限

- 政府接口仅返回最近一个交易日，历史需每日 `--merge` 积累
- 广州二手房暂无稳定日更 API（官网为月度图片公告），App 中广州仅显示新房
