# v0.5.0 — 行政标准化 + 学校扩充（Option A）

**日期**：2026-07-12
**类型**：数据 / 测试
**前置**：v0.4.3

## 背景

Option A：拉政府/权威开源开放数据。  
评估 `opendata.sz.gov.cn` 后发现其 API 接入需要 appkey 注册（流程 ≥ 3 工作日），暂缓。  
改用公开开源数据集做"政府开放数据等价物"：

- `modood/Administrative-divisions-of-China` → 行政区标准化
- `leiii/census` → 县区人口（备用）

加上手填的知名中小学，覆盖教育板块的"政府开放数据"语义。

## 改动

### 数据
- **新增** `static/seed/admin_districts.csv`
  - 23 条官方区（广州 11 + 深圳 9 + 珠海 3）
  - 字段：`city_id`, `city_code`, `district_code`, `district_name`
  - 人工补 1 条：`大鹏新区`（非民政部标准名但深圳惯用，已记录）

- **扩充** `static/seed/schools.csv`：**14 → 58 条**（+44）
  - 深圳 32 / 广州 18 / 珠海 8
  - 字段：`school_id, school_name, city_id, district_name, school_type, is_key, address, lat, lng`

- **扩充** `static/seed/school_indicators.csv`：**14 → 58 条**（一一对应）

### 脚本
- **新增** `scripts/import_admin_divisions.py`
  - 拉 `modood/Administrative-divisions-of-China` 的 `areas.json`
  - 过滤 3 城 → 输出 `admin_districts.csv`
  - 纯 stdlib：`urllib` + `json` + `csv`
- **新增** `scripts/validate_districts.py`
  - 校验 `communities.csv` 的 `district_name` 是否在 `admin_districts.csv` 内
  - 不匹配 → 打印 + exit 非 0
- **新增** `scripts/seed_schools.py`
  - 手填数据 + 自动生成对应 `school_indicators.csv`
  - 保证 `schools.length === school_indicators.length`

### 测试
- `tests/buildIntegrity.test.ts`
  - 新增 describe `政府开放数据配套（v0.5.0 / Option A）`
  - 5 条用例：`admin_districts.csv` 存在 / ≥23 行 / `schools ≥ 50` / `schools === indicators` / `communities.district_name` 全在 admin 里
- `tests/e2e/smoke_admin.mjs`（新）
  - 浏览器拉 3 城 csv，校验"南山区 / 大鹏新区 / ≥50 schools / schools==indicators"
  - 跑 3 个页面截图：dashboard / listings / 深圳主页
