# v0.41.0 · 房源新鲜度 (trend-21)

**主题**：基于 `listings.crawl_date` 算每小区的"挂牌活跃度"，区分"新挂牌多"和"滞销"两类小区。

## 数据源
- `static/seed/listings.csv` (1286 行, 含 `crawl_date`)
- `static/seed/cities.csv` (city_name)
- `static/seed/communities.csv` (community_name + district_name)

## 公式
```
freshness_score = min(100, (近 4 周 × 1 + 近 2 周 × 2) ÷ 总数 × 100)

thresholds (新挂牌榜颜色):
  >= 30: 🟢 活跃
  >= 15: 🟡 中等
  < 15:  🔴 滞销

median_age_days = median(today - crawl_date) for each listing
stale_count = listings where age > 84 days
```

排除 total_listings < 5 的小区 (样本过小，freshness=100 噪声)。
