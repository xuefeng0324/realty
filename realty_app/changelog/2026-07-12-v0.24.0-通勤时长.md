# v0.24.0 — 通勤时长榜 (new-5)

## 数据源
- 高德 `/v3/direction/transit/integrated` — 公交路径规划 API
- 目的地：城市核心 CBD
  - 深圳 → 福田CBD (深圳福田CBD逸扉酒店, 114.0508, 22.5422)
  - 广州 → 珠江新城 (113.3212, 23.1194)
- 调用时机：早 08:30 (通勤高峰)
- 共 38 次 API 调用 (深圳 30 + 广州 8)

## 改动
- 新增 `scripts/crawl_amap_commute.py` (CBD POI 搜 + transit API + fallback walking)
- 新增 `static/seed/commute.csv` (38 行)
- 新类型 `LocalCommute` (community_id, city_id, city_name, cbd_name, cbd_lat/lng, transit_minutes, transit_distance_m)
- 新 query: `getCommuteRanking({ cityId, limit })` → `CommuteRankingResponse` (fastest[], cityAvgMinutes, totalCommunities)
- 新 store: `getCommutes()`, `getCommutesByCity()`, `getCommuteByCommunity()`
- dashboard 新增「🚇 通勤时长榜」卡
- 数据源标签：transit/integrated 公交方案 1 + 早 08:30

## UI
- 卡标题: "🚇 通勤时长榜 · {city} → {CBD}"
- 右上: 城市均 N 分钟 · N 小区
- 每行：rank medal + 小区名 + 区名 (muted) + 分钟 badge (绿/灰/红) + km 距离
- 行可点 → 小区详情
- 边际保护：
  - 比均值快 15% → price-down (绿)
  - 接近 → muted (灰)
  - 比均值慢 30% → price-up (红)
