# v0.25.0 户型/面积/朝向/装修分布 (new-7)

## 新功能

dashboard 新增「🏠 户型分布 · {城市}」卡片，按 4 个维度统计当前城市的在售房源分布：

- **户型** (bedrooms): 1室 / 2室 / 3室 / 4室 / 5室+ / 未知
- **面积 (㎡)** (area_sqm): <50 / 50-80 / 80-110 / 110-150 / 150+ / 未知
- **朝向** (orientation): 南 / 东南 / 南北通透 / 西南 / 西 / 东 / 北 / 西北 / 东北 / 其他 / 未知
- **装修** (decorate): 精装 / 豪装 / 普装 / 简装 / 毛坯 / 其他 / 未知

每个 bucket 显示：bucket 名称、条形比例 (share)、房源数、占比 (含 1 位小数)。

## 数据源

- `realty_app/scripts/compute_layout_distribution.py` — 按 (city_id, dimension, bucket) 聚合 `listings.csv`
- `realty_app/static/seed/layout_distribution.csv` — 54 行 (3 城市 × 4 维度 × 多个 bucket)
- 字段：`city_id, city_name, dimension, bucket, count, share, median_unit_price, avg_area_sqm`

## 数据流

1. **importer.ts** 解析 `layoutDistributionCSV` → `LocalLayoutDistribution[]`
2. **store.ts** 提供 `getLayoutDistributions()` / `getLayoutDistributionsByCity()`
3. **queries.ts** 提供 `getLayoutDistribution({ cityId })` → `LayoutDistributionResponse`
4. **dashboard.vue** 通过 `loadAll()` 调用，渲染 4 维度分布卡片
