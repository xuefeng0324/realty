# v0.23.0 — 全品类区级网签热度榜 (trend-9)

## 改动
- dashboard 新增「🔥 全品类区级网签热度榜」卡（在 v0.10.0「近 4 周二手网签热度榜」之后）
- 3 tab 切换：**新房** / **二手** / **全部** (默认全部)
- 列字段：区名 + 累计套数 + 面积 + 套/天
- 显示 Top 10 (总数 = totalDistricts)
- 数据源：wangqian_district_weekly.csv (深圳 44 / 广州 22 行，新房 44 / 二手 22)

## 新 query
- `getDistrictWangqianRank({ cityId, category, weeksBack, limit })`
- 返回 `DistrictWangqianRankResponse` (含 totalUnits, totalAreaSqm, totalDistricts, items[])
- items[] 字段：rank, district, totalUnits, totalAreaSqm, weeks, avgDailyUnits, avgUnitAreaSqm
- 复用 `store.getWangqianTopDistricts`，limit 设为 999 后再 .slice(0, limit)

## UI
- 卡标题：「🔥 全品类区级网签热度榜 · {cityName}」
- 右上：累计 N 套 · N 区
- tab: 新房 / 二手 / 全部 (高亮蓝，off 透明)
- 行：rank medal + 区名 + 横柱 + 「N 套 (X m² · N套/天)」
- 边际保护：items.length > 10 时显示「共 N 个区，显示 Top 10」
