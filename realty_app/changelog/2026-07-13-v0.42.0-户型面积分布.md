# v0.42.0 · 户型 × 面积 联合分布 (trend-22)

**主题**：2D 热图展示 (户型 × 面积) 桶的房源数量 + 中位单价，让用户一眼看出"哪类户型集中在哪面积段"。

## 数据源
- `static/seed/listings.csv` (1286 行, `bedrooms` + `area_sqm`)

## 公式
```
area_bucket = if area<50: "<50"
           elif area<80: "50-80"
           elif area<110: "80-110"
           elif area<150: "110-150"
           elif area<200: "150-200"
           else: "200+"

per (city, bedrooms, area_bucket):
  count = listings 命中数
  share = count / city_total
  median_unit_price = median(prices)
```

minCount=3 过滤掉小样本 (cell 显示 "—")

## dashboard 新卡
**「📐 户型 × 面积 分布 · {city}」**:
- 5 行 (1/2/3/4/5 室) × 6 列 (6 面积桶) = 30 cells
- 每 cell: 上=套数, 下=中位单价 (千元/㎡)
- 颜色深度: cell opacity = count / max(count) (min 0.15)
- 空 cell 显示 "—"
- 蓝渐变 (浅蓝=少, 深蓝=多)
