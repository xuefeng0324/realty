# v0.30.0 trend-14 区涨幅榜

## 新功能

dashboard 新增「🚀 区涨幅榜 (近 4 周) · {城市}」卡片，按最近 4 周累计中位价变化排序。

- 显示：排名、区名、最新 WoW (周环比)、4 周累计变化
- 颜色编码：>+0.5% 红 / <-0.5% 绿 / 其他灰
- 金牌前 3 (medal-gold / silver / bronze)

## 数据流

1. **getDistrictChangeRank({ cityId })** query：
   - 每个区取最近一周和 4 周前的 medianUnitPrice
   - `recentChange4w = (last - 4w_ago) / 4w_ago × 100%`
   - 按 `recentChange4w` 降序排序
2. dashboard 「🚀 区涨幅榜」卡 UI
