# v0.52.0 — map-12 行政区域图 5 种叠加模式

地图 tab 内的「行政区域图」卡片增加 5 种模式切换 (📍 社区 / 🔢 小区数 / 💰 均价 / 🏫 学区 / 🚇 地铁), 复用 listings / listingSchoolPremium / metroWalk 三类已聚合数据, 不增加新 CSV, 实时从 store 计算.

## 变更

### 1. `src/pages/dashboard/dashboard.vue` — 5 mode 切换
新增 `MAP_MODES` (5 项) + `mapMode` ref (默认 `marker`) + 顶部 tab 按钮组 (`map-mode-tabs`) + 颜色梯度图例 (`map-legend`).

5 种模式对应不同 fill 颜色 / 显示内容:

| 模式 | icon | label | 颜色梯度 | 数据源 (区聚合) |
|---|---|---|---|---|
| `marker` | 📍 | 社区 | 浅灰底 | 当前 marker 显示 (圆点 + 名字) |
| `count` | 🔢 | 小区数 | 浅蓝→红 (越高越深) | `store.getListingsByCity` (unitPrice > 0) → 各区 count |
| `price` | 💰 | 均价 | 浅蓝→红 (越高越深) | `store.getListingsByCity` → 各区 avg unitPrice |
| `school` | 🏫 | 学区 | 浅蓝→红 (越高越深) | `store.getListingSchoolPremiumByCity` → 各区 avg avgSchoolScore |
| `metro` | 🚇 | 地铁 | 绿→红 (越低越好, **反转**) | `metroWalk.items` → 各区 avg walkMinutes |

### 2. 颜色计算
新增 `mapDistrictStats` computed (聚合 4 项指标到 districtName), `districtFill(name)` 返回颜色 hex, `mapStatRange` computed 计算 min/max, `mapModeGradient` 返回 CSS gradient 字符串.

- count/price/school: 5 档色阶 (`#bae6fd → #86efac → #fde047 → #f97316 → #ef4444`)
- metro: 4 档反转色阶 (`#86efac → #fef08a → #fca5a5 → #ef4444`, 越低越绿)

### 3. UI 调整
- 非 marker 模式: marker `<g>` 隐藏, 改为在每个区中心显示该区聚合值文字 (`map-district-val`, 浅色描边突出).
- 顶部 tab: 5 个等宽 tab, 激活态用紫渐变.
- legend: 标题 + 渐变条 + min/max 数字, 跟着模式动态更新.

### 4. 测试
- `tests/buildIntegrity.test.ts` 新增 4 个 v0.52.0 测试:
  - `MAP_MODES` 含 5 模式 + `mapMode = ref<MapModeKey>("marker")`
  - 5 个 `.map-mode-tab` + `v-for="m in MAP_MODES"` + `@click="mapMode = m.key"`
  - `districtFill` + `districtStatLabel` + `mapStatRange` + `mapDistrictStats` 都已定义
  - 非 marker 模式隐藏 marker, 显示 `map-district-val`, 使用 `:fill="districtFill(d.districtName)"`
