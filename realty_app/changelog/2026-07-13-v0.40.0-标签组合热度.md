# v0.40.0 · 标签组合热度 (trend-20)

**主题**：用 listing_tags.csv 的 4-7 个 tag 做 **2-combination (pair) 频率统计**，找出"最常一起出现的标签组合"。

## 数据源
- `static/seed/listing_tags.csv` (7518 行)：每个 listing 4-7 个 tag
- `static/seed/listings.csv` (1286 行)：city_id + unit_price
- `static/seed/cities.csv` (3 行)：city_name lookup

## 公式
对每个 listing 取 sorted tags, C(n, 2) 算 2-组合 pair, 跨 listing 累计 count。过滤 count ≥ 5。
