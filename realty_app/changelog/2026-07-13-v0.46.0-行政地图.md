# v0.46.0 · 行政区域地图 + 社区 marker (map-11)

**主题**: 第一张真正的"地图"视图 —— 在 SVG 内渲染 city 全部 district 多边形 + 全部 community marker + 区名/社区名 label。

## 数据源
- **高德 /v3/config/district**: 24 行政区 polygon (3 city × ~8 区)
  - 481 KB JSON, 5474 多边形 ring 总点数 (平均 274 pts/ring)
  - 含洞处理: fill-rule=evenodd, 海珠区/天河区等多环 polygon 正确显示
- **`static/seed/communities_geo.csv`**: 51 社区经纬度 (v0.4.1 已有)

## 新增产物
- `scripts/crawl_district_polygon.py`: 高德 API + 增量缓存 (重复运行不重复拉)
- `static/seed/district_polygon.csv`: 24 区 polygon + center
- `static/seed/communities_geo.csv`: 已有数据
- `src/local/types.ts`: `LocalDistrictPolygon` interface + `communityGeo` array
- `src/local/importer.ts`: `parseDistrictPolygon(cityCodes)` + `parseCommunityGeo`
  - cityCodes Map<cityId, city_code> 用于从 district_code 推断 cityId
  - cityCode 前 4 位匹配 (4401=广州, 4403=深圳, 4404=珠海)
- `src/local/store.ts`: `getDistrictPolygon()` / `getDistrictPolygonByCity()` / `getCommunityGeo()` / `getCommunityGeoByCity()`
- `src/local/queries.ts`: `getDistrictMap(cityId)` → bbox + polygons + markers
  - bbox 自动算 (5% padding)
- `src/pages/dashboard/dashboard.vue`: 新卡「🗺️ 行政区域图 · {city}」

## dashboard 新卡

**「🗺️ 行政区域图 · {city}」**:
- Header 显示 `{N} 区 · {M} 社区`
- **Inline SVG (660×480 viewBox, fill-rule: evenodd for holes):**
  - 行政区: `<path>` 用 `d` 属性串联 ring 外环 + 内环, hover 变色 (深蓝/橙色)
  - 区名 label: `<text>` 在 district centerLng/Lat, 蓝色加粗 + 白光晕
  - 社区 marker: `<circle>` 红色填充 5px 半径
  - **≤30 社区时**: 显示 marker + 名字 (`<text>` 偏移 +7px)
  - **>30 社区时**: 简化为 bare marker (3px 红点)
- 灰底 + 4px radius
- 高德风浅渐变 (后改纯灰 `#f1f5f9` 提升对比)

## 关键实现
- **lng/lat → SVG 坐标**: lat 反向 (SVG y 向下增长)
- **ring → path d**: `M x1,y1 L x2,y2 ... Z` 
- **填充规则**: `fill-rule="evenodd"` 自动处理多环 polygon (内环 = 洞)
- **bbox padding**: 5% 自动避免边缘裁剪
- **communityGeo cityId**: csv 中无 city_id, 但 community_id 全局唯一, 已有 listings/community join
- **cityId 推断**: district_code 前 4 位 匹配 city_code 前 4 位

## 业务价值

- 社区 marker 显示密度 = 哪区房源最多
- 视觉震撼: 24 个真实 polygon 不是 placeholder, 高德数据
