# v0.55.0 — hero-1 首页 hero 轮播 + 快捷入口

Dashboard 第一页 (顶部 sticky 周次 + tabs 之后, 第一张卡片之前) 新增:
1. **顶部大盘轮播**: 横向滑动 6 张城市级快照卡 (总挂牌 / 中位单价 / 中位总价 / LPR / 通勤 / 学区)
2. **快捷入口图标网格**: 4×2 网格 8 个圆形彩色图标, 一键直达 (4 tab + 切换城市 + 切换周次 + 数据设置 + 小区榜单)
3. **圆点指示器**: 6 个圆点, 当前激活的拉长为紫色胶囊

## 设计思路 (借鉴 Airbnb / Beike / 小红书首页)

- 用户打开 dashboard 不必滚到第一个具体卡片就能看到大盘状态
- 快捷入口图标网格替代传统 "列表式导航", 节省 50% 纵向空间
- 圆点 + auto-play 让用户感知 "还有更多"
- 点击 hero 卡直接跳到对应 tab, 缩短用户路径

#### 1.3 自动滚动

```ts
let heroTimer: number | null = null;
function startHeroAuto() {
  if (heroTimer != null) return;
  heroTimer = setInterval(heroAdvance, 5000) as unknown as number;
}
function stopHeroAuto() {
  if (heroTimer != null) { clearInterval(heroTimer); heroTimer = null; }
}
onMounted(async () => { ... await loadAll(); startHeroAuto(); });
onUnmounted(() => stopHeroAuto());
```
