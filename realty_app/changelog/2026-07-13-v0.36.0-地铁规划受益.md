# v0.36.0 地铁规划受益

> 2026-07-13

## 🎯 这次改了什么

新增 **"🚇 地铁规划受益 Top"** 卡片 — 展示每个小区到**未来 1-5 年即将开通的地铁站**的距离，按距离 × status 综合打分排序。

这是 v0.35.0 (步行到**现有**地铁) 的姊妹篇 — 现在用户能同时看到：
- 现在去哪都方便（v0.35.0）
- 未来 3-5 年**还会更方便**（v0.36.0）

## 🆕 新增

- `scripts/compute_metro_benefit.py`：纯本地脚本（无 API 调用），对 49 小区 × 21 规划线路
  - 把 `metro_planning.csv` + `metro_planning_geo.csv` join，提取 42 个 start/end 站
  - haversine 算 小区 → 每站 距离 → 受益分（`≤300m:100 / ≤500m:90 / ≤800m:75 / ≤1200m:60 / ≤2000m:40 / ≤3000m:20 / >3000m:0`）
  - 乘 status 权重：
    - **即将开通** × 1.5 (板上了)
    - **在建** × 1.2 (大概率按期)
    - **规划** × 1.0 (≤2030) / × 0.7 (>2030，预测可信度低)
  - 同分时按 距离 升序 (tiebreak)
  - **必须同城**（s.city_id != c.city_id 跳过，避免误匹配）

- `static/seed/metro_benefit.csv` (49 行，3 城市)：
  - 深圳 30 行：top3 都是 **72 分**（中海天钻/星河智荟/诗宁别墅/聚福大厦/科技楼 → 17/21/19 号线一期）
  - 广州 11 行：top1 **保利天悦 90 分** (8号线东延 791m)
  - 珠海 8 行：top 0 分（南珠/珠肇高铁站距离都 >3km）

- `LocalMetroBenefit` 类型 + `parseMetroBenefit` 导入
- `store.getMetroBenefits()` / `getMetroBenefitsByCity(cityId)`
- `queries.getMetroBenefitRanking({ cityId, topN, minScore })`：返回 avg/max/**nearCount** (≥60 的小区数)
- Dashboard "🚇 地铁规划受益 Top · {city}" 卡片：
  - 每行：`受益分 / 区 · → 几号线「站点」/ status 徽章 / 距离m + 开通年份`
  - 受益分 3 档：🟢 ≥75 / 🟠 ≥40 / 🔴 0-39
  - status 徽章 3 色：在建(橙) / 即将开通(绿) / 规划(灰)

## ⚙️ 为什么这个数据重要

买房决策"未来 3-5 年"的地铁变化是**隐性增值**：

- 现在偏远的地方，可能因为**规划地铁站**而大幅增值
- 比如深圳光明 / 龙岗 / 坪山，原本没有 5 号线 / 3 号线，2026-2028 多条地铁线开通后通勤到 CBD 时间砍一半
- v0.36.0 让用户能看到"我未来 5 年会有什么新地铁，升值空间多少"
