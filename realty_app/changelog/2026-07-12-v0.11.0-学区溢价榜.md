# v0.11.0 - 学区溢价榜

> 2026-07-12

## 一句话

dashboard 新增「学区溢价榜」卡片：把每区的"学区评分"与"挂牌均价中位数"关联，告诉你哪些区因名校聚集产生了正向溢价。

## 数据流

```
schools.csv (58 个学校，新增 district_name 列)
       │
       ▼
school_indicators.csv (latest_level_score_raw)
       │
       ▼ 聚合
按 (city_id, district_name) 平均 → 区的 school_score
       │
       ▼
listings.csv (1286 条) → 按 (city_id, district_name) 算中位单价
       │
       ▼
premium_ratio = (区 median / 全市 median - 1)
       │
       ▼
school_premium_district.csv   (16 行 × 3 城)
school_premium_community.csv  (52 行)
```

## 新增 / 改动

### 数据脚本

- `scripts/enrich_school_districts.py`：给 `schools.csv` 加 `district_name` 列（58 条全部基于公开位置信息手填）。
- `scripts/compute_school_premium.py`：从 listings + schools + school_indicators 聚合出板块/小区级学区评分。

### 数据文件

- `static/seed/schools.csv`：新增 `district_name` 列。
- `static/seed/school_premium_district.csv`（新增，16 行）。
- `static/seed/school_premium_community.csv`（新增，52 行）。

### 数据层

- `src/local/types.ts`：`LocalSchoolPremiumDistrict` / `LocalSchoolPremiumCommunity` 接口。
- `src/local/importer.ts`：新增 `schoolPremiumDistrictCSV` / `schoolPremiumCommunityCSV` 输入。
- `src/local/seedSnapshot.ts`：默认加载两份 CSV。
- `src/local/store.ts`：`getSchoolPremiumDistricts` / `getSchoolPremiumRank` / `getCommunitySchoolScore`。
- `src/local/queries.ts`：`getSchoolPremiumRank(cityId)`。
- `src/local/dataRefresher.ts`：保留学区数据。
- `src/pages/settings/settings.vue`：`csv-url` 模式拉取。

### UI

- `src/pages/dashboard/dashboard.vue`：新增「学区溢价榜」卡片，按 `premium_ratio` 降序展示 Top 区。
  - 金/银/铜牌 + 评分 + 溢价% + 中位单价
  - 正溢价绿色，强溢价深绿，负溢价红色
