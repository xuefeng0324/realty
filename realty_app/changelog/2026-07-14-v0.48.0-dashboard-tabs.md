# v0.48.0 dashboard-tabs 顶部 Tab 分类

**release date:** 2026-07-14

**summary:** dashboard 已经堆到 26 张卡, 用户翻起来很累。加一个顶部 5-tab 切换, 按场景分门别类:

## 5 个 Tab

| Tab | icon | 卡片数 (深圳样例) | 涵盖 |
|-----|------|----|------|
| 📊 **全部** | 📊 | 34 | 所有卡 (默认) |
| 💰 **价格画像** | 💰 | 15 | 区房价指数 / 区涨幅榜 / 综合评分 / 朝向楼层 / 装修楼龄 / 户型×面积 / 散点 / 特征画像溢价 / 标签组合热度 / listing 溢价 / 户型分布 / 通勤 |

| 🚇 **通勤地铁** | 🚇 | 5 | 通勤榜 / 地铁步行 / 地铁规划 / 商业热度 / 生活便利度 |
| 🗺️ **地图视图** | 🗺️ | 2 | 行政区图 (district map) / 社区散点 |

## 实现机制

`data-tab` 属性 + 全局 CSS:

```scss
body[data-dash-tab="price"] .card:not([data-tab*="price"]),
body[data-dash-tab="school"] .card:not([data-tab*="school"]),
body[data-dash-tab="transit"] .card:not([data-tab*="transit"]),
body[data-dash-tab="map"] .card:not([data-tab*="map"]) {
  display: none !important;
}
```

每张卡声明 `data-tab="all,price"` (逗号分隔多个), 切换 active 时
JS 设置 `document.body.setAttribute("data-dash-tab", k)`, CSS 自动隐藏。

## UI 设计

- 卡片容器: 圆角 16rpx, 淡灰渐变背景
- 单 tab: 140rpx 宽, 32rpx icon + 24rpx label
- active tab: 紫蓝渐变 `#6366f1 → #8b5cf6`, `scale(1.03)` 抬起
- 横向 overflow-x: auto (手机端可滑动)

## 卡片分类约定

按"用户的购买决策场景"分, 而不是数据维度:
- **价格画像**: 谈价 / 比价 / 找特征
- **学区配套**: 落户 / 找学区 / 看配套
- **通勤地铁**: 上班 / 出行 / 看交通
- **地图视图**: 看地理 / 找位置
