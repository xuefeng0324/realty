# v0.56.0 — detail-2 community 详情页 UX

延续 v0.54.0 listing 详情页 UX 模式: community 详情页同样加顶部快捷导航 + 同区其他小区横向对比.

## 变更

### 1. `src/pages/community/community.vue` — 顶部快捷导航

| 按钮 | 行为 |
|---|---|
| ← 返回 | navigateBack → fallback switchTab dashboard |
| 📊 仪表盘 | switchTab → dashboard |
| 🗺️ 地图视图 | switchTab → map-view (新) |
| 🏘️ 同区其他 (N) | 主按钮 (紫渐变), 显示同区其他小区数 |

### 2. 同区其他小区卡片
新增「🏘️ 同区其他小区 · {district}」卡片:
- 数据源: `getCommunitiesByCity(cityId)` 按 `districtName` 过滤
- 排除当前社区自身
- 实时从 listings 聚合: 中位单价 / 在售套数 / 楼龄 min-max
- 按 `medianUnitPrice` 降序排序, 取 Top 10
- 每行 `tap-row` hover + `@click="goCommunity(id)"` → `uni.redirectTo`
- 底部提示: 楼龄 / 户型 / 物业差异通常解释 20-30% 单价差

### 3. 新增导航函数
```ts
function goBack() { /* 智能回退 */ }
function goDashboard() { /* switchTab */ }
function goMap() { /* switchTab map-view, 新 */ }
function goCommunity(id) { /* redirectTo 同区其他小区 */ }
```

## 数据局限

- 同区其他小区数量取决于 snapshots 中同一 districtName 的社区数 (广州 4 区平均 ~6 社区, 深圳 9 区平均 ~5 社区)
- 楼龄 min/max 用 `buildYear` 字段 (Listings 字段), 部分 listings 缺失显示 null
- 中位单价实时从 `getListingsByCommunity` 聚合, 不需要新 CSV

## UX 改进

- 用户从 listing 跳到 community 详情页后, 不必返回 dashboard 才能看其他小区
- 4 个快捷按钮 + 同区横向对比 = 形成完整浏览路径
- 复用 v0.54.0 模式, 代码一致性高, 维护成本低
