## 1. 背景

v0.31.0 生活便利度覆盖 5 类 POI (mall/park/subway/school/hospital)，但**菜市场** (fresh market) 是中国城市居民日常生活高频配套 (俗称"买菜便利度")，原 POI 列表没有，缺失。

本次 v0.32.0:
1. 新增 `crawl_market_poi.py` 抓取 49 个小区周围 1.5km 内的菜市场 POI (高德 `/v3/place/around`, 4 keywords)
2. 输出 `poi_market.csv` (147 行 / 49 小区 = 3 POI each)
3. 升级 `compute_life_convenience.py` 加 **菜市场** 维度 (100 → **110 满分**)
4. 升级 `life_convenience.csv` schema: 新增 `market_near` + `score100` (归一化)
5. 升级 dashboard 卡: 6 维展示 (M/P/S/X/Y/**C**), 颜色分档用归一化 score100

## 2. 数据源

## 4. 升级打分规则 (满分 110)

| 维度   | ≤500m | ≤800m | ≤1000m | ≤1500m | ≤2000m | ≤3000m | max |
|--------|-------|-------|--------|--------|--------|--------|-----|
| 商场   | 25    |       | 20     | 10     |        |        | 25  |
| 公园   | 15    |       | 10     | 5      |        |        | 15  |
| 地铁   | 25(≤300) | 15 | 20(≤500) | 5    |        |        | 25  |
| 学校   | 10    |       | 7      | 4      |        |        | 10  |
| 医院   |       |       | 15     |        | 10     | 5      | 15  |
| **菜市场** | **20** |   | **15** | **10** |        |        | **20** |

总 max = 25+15+25+10+15+20 = **110**

## 5. 升级 schema

`life_convenience.csv` 新增字段:

| 字段 | 说明 |
|------|------|
| `market_near` | 菜市场维度得分 (0-20) |
| `score` | 综合分 (0-110) |
| `score100` | 归一化分 (0-100) = score / 110 × 100 |

## 6. UI 升级

dashboard 「🧭 生活便利度 Top 小区」卡:
- 6 维评分 (M/P/S/X/Y/**C**), 加 C=菜市场 维度
- 城市均分/最高分显示 `/ 110`
- 单小区总分显示 **score100 (归一化 0-100)**, 颜色分档不变 (≥80 绿 / 60-79 蓝 / <60 灰)
