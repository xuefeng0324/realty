# v0.6.0 — 医院清单 + UI 集成

**日期**：2026-07-12
**类型**：数据 / UI
**前置**：v0.5.0

## 背景

之前 POI 周边配套的"医院"类只是高德周围搜索（poi_seed.csv 的 1.5km 半径），缺少：
- 医院等级（三甲 / 三级 / 二甲）
- 医院类型（综合 / 中医 / 妇幼 / 专科）
- 行政区定位
- 城市级医院清单

## 改动

### 数据
- **新增** `static/seed/hospitals.csv`：**50 条**
  - 深圳 25 / 广州 19 / 珠海 6
  - 字段：`hospital_id, city_id, official_name, display_name, hospital_type, hospital_level, district_name, address, lat, lng, key_flag`
  - 三甲 35 条，二级 15 条
  - 数据源：本地宝 + 各市卫健委公开目录（手填）

- **新增** `static/seed/hospitals_geo.csv`（高德 POI 校验输出）
  - 字段：`hospital_id, lat, lng, amap_poi_id, formatted_address, confidence, source, distance_m`
  - 50/50 行；confidence：high=4 / medium=24 / low=22

- **扩半径** `crawl_amap_poi.py`：hospital 类半径从 1500m 扩到 **3000m**
  - `poi_seed.csv` hospital 类 113 → 143 行

### 脚本
- **新增** `scripts/seed_hospitals.py`（手填 50 条）
- **新增** `scripts/crawl_amap_hospital.py`（高德 POI 校验 + 距离打分）

### 代码层
- `src/local/types.ts` — 加 `LocalHospital`
- `src/local/importer.ts` — 解析 `hospitalsCSV` → `hospitals: LocalHospital[]`
- `src/local/seedSnapshot.ts` — 默认注入 hospitalsCSV
- `src/local/store.ts` — 加 `getHospitals/getHospitalsByCity/getHospitalById/getHospitalsByDistrict`
- `src/local/queries.ts` — 加 `getCommunityHospitals`（5km 内 + 同区兜底）
- `src/local/dataRefresher.ts` — refresh 时保留 hospitals
- `src/pages/listing-detail/listing-detail.vue` — 新增 "周边医院" 卡片（5 类色码等级标签）
- `src/pages/community/community.vue` — 同上
- `src/pages/settings/settings.vue` — csv-url 模式拉 hospitals.csv
